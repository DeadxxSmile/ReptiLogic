# ReptiLogic Static Site

This is a simple static landing page prepared for GitHub Pages or any other static host.

## Files
- `index.html`
- `styles.css`
- `assets/logo/reptilogic-logo.png`
- `assets/screenshots/*`

## GitHub Pages quick setup
1. Create a repo for the site or use your existing project repo.
2. Upload these files into the repo root, or into `/docs` if that is how you prefer to publish.
3. In GitHub, open **Settings > Pages**.
4. Set the source to the correct branch and folder.
5. If you are using a custom domain, add it in the Pages settings and update your DNS records.
